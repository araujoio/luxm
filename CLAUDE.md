# Luxm Project

This project was generated with [Luxm](https://github.com/araujoio/luxm)

## What is Luxm?

Luxm is a command-line tool that automates the creation of Next.js projects with support for multiple libraries, eliminating repetitive setup tasks and letting developers focus on what actually matters building the product.

Beyond scaffolding, Luxm leverages AI-powered skills and MCPs (Model Context Protocols) under the hood to automate complex workflows like bulk-translating pages, managing routes, and maintaining project consistency. This means less manual config and more productivity, right from your terminal.
